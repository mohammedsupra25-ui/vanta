# VANTA Project

## Stack
- React 18 + TypeScript + Vite + Tailwind CSS v3
- Fonts: Cormorant Garamond (display) + Syne (UI)
- Animations: GSAP + ScrollTrigger, Framer Motion, Lenis smooth scroll
- Particles: @tsparticles/react + @tsparticles/slim
- Color system: pure black (#000000) to white (#ffffff), zero colors
